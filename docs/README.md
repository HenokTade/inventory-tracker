# Inventory Tracker System

## Overview
A simple web-based inventory tracking system built with Flask that allows users to manage inventory items through a clean, intuitive interface.

## Features
- **User Authentication**: Secure login system with session management
- **Dashboard View**: Display all inventory items in a organized table
- **Add Items**: Quick form to add new inventory items with name and quantity
- **Data Persistence**: All data stored in JSON format for easy access

## Technology Stack
- **Backend**: Python Flask
- **Frontend**: HTML, CSS
- **Data Storage**: JSON file-based storage
- **Session Management**: Flask sessions
